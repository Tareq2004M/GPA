## Grading Scale Classifications
### 5.00 GPA System:
* **4.75 - 5.00:** High Excellent 
* **4.50 - 4.74:** Excellent 
* **3.75 - 4.49:** Very Good
* **2.75 - 3.74:** Good
* **2.00 - 2.74:** Pass
* **Below 2.00:** Fail 

### 4.00 GPA System:
* **3.75 - 4.00:** High Excellent 
* **3.50 - 3.74:** Excellent 
* **2.75 - 3.49:** Very Good
* **2.00 - 2.74:** Good
* **1.00 - 1.99:** Pass
* **Below 1.00:** Fail
